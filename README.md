# 🚀 SmartEvent AI Navigator System

> An intelligent, AI-inspired crowd management and navigation system designed to optimize event experiences through real-time insights, predictive analysis, and smart decision-making.

👉 https://sushmita25dubey.github.io/GOOGLE-PromptWars-Challenge-1/

## 📌 Problem Statement

Large-scale events such as stadiums, concerts, and festivals often face:

* 🚶 Crowd congestion
* ⏳ Long waiting times
* 📡 Lack of real-time coordination

These challenges reduce attendee satisfaction and create safety risks for event organizers.

---

## 💡 Solution Overview

**SmartEvent AI Navigator System** is a lightweight, intelligent web application that simulates real-time crowd behavior and provides **smart, data-driven recommendations**.

It serves both:

* 🎯 **Attendees** → Helps choose best gates and routes
* 👨‍💼 **Organizers** → Provides insights for crowd control

---

## 🧠 Key Features

### 🔹 Real-Time Crowd Monitoring

* Displays live crowd levels (Low / Medium / High)
* Continuously updates using realistic simulation

---

### 🔹 ⏳ Waiting Time Estimation

* Low → ~2 min
* Medium → ~5 min
* High → ~10 min

---

### 🔹 🎯 Smart Recommendation Engine

* Recommends the best gate based on:

  * Crowd level
  * Duration of congestion
* Avoids gates that are heavily crowded for long periods

---

### 🔹 ⏱ Time-Based Crowd Intelligence (Core Innovation)

* Tracks how long each gate remains in a state
* Detects **persistent congestion**
* Enables smarter and more reliable decisions

---

### 🔹 🔮 Crowd Prediction

* Predicts possible future congestion
* Alerts users in advance

---

### 🔹 🧭 Smart Navigation Guidance

* Suggests optimal routes using least crowded paths

---

### 🔹 🚨 Real-Time Alerts & Updates

* Dynamic updates about congestion and clearance

---

### 🔹 👨‍💼 Management Insights

* Identifies congested gates
* Tracks congestion duration
* Supports real-time crowd control decisions

---

## ⚙️ How It Works

1. Simulated real-time crowd data is generated
2. Each gate tracks:

   * Crowd level
   * Duration in that state
3. Intelligent logic:

   * Analyzes patterns
   * Predicts congestion
   * Recommends optimal gates
4. Chatbot interface enables interactive assistance
